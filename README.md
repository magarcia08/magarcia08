## 📂 Proyectos destacados

### 🏢 TalentX — Sistema de Gestión de RRHH
> Sistema fullstack para la administración integral del talento humano en una organización.

- Módulos completos de empleados, departamentos, asistencias y nómina
- Backend dual con **Node.js/Express** y **Spring Boot** conectados a **MySQL 8**
- Autenticación con roles Admin/Empleado y cifrado de contraseñas con bcrypt
- Frontend responsivo con HTML, CSS y Bootstrap

`Java` `Spring Boot` `Node.js` `MySQL` `HTML` `CSS` `Bootstrap`

---

### 📚 BiblioTrack — Gestor de Biblioteca
> Plataforma web para la gestión de libros, usuarios y préstamos en una biblioteca.

- Esquema relacional de 6 tablas con migraciones en MySQL
- API RESTful para CRUD completo de libros, usuarios y préstamos
- Interfaz limpia y responsiva con Bootstrap

`Node.js` `MySQL` `HTML` `CSS` `Bootstrap`

---

### 🎮 ABC Aventura AR — Videojuego Educativo
